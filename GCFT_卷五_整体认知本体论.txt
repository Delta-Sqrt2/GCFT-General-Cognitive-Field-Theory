14.1  价值问题的真实缺口

    为什么"空性态"是"好"的？为什么 g → cI 是演化的"目标"而
    非任意的稳态？

13.0 方案的正确直觉是：需要一个"价值"概念独立于痛苦。但 13.0
原方案将价值 $\mathcal{V}$ 作为新公理引入，违反卷一原则一

    价值不是新公理，价值是度规的 von Neumann 熵的负值。

14.2  价值势能场的严格定义

定义 14.1（价值势能场）
设度规 g 的本征值为 $(\lambda_1, ..., \lambda_d)$，在共形约束
$\sum_i \lambda_i = d$ 下，归一化概率 $p_i = \lambda_i / d$。
价值势能场定义为标准 von Neumann 熵：

    $\mathcal{V}(g) := -\sum_i p_i \log p_i \geq 0$

其与卷一作用量中的谱曲率项 $N_{GCFT}(g) = \sum_i \lambda_i \log
\lambda_i$ 的关系为：

    $\mathcal{V}(g) = \log d - N_{GCFT}(g)/d$

证明：由 $\lambda_i = d \cdot p_i$ 代入 $N_{GCFT}(g)$：
    $N_{GCFT}(g) = \sum_i (d p_i) \log(d p_i) = d \log d + d \sum_i
    p_i \log p_i = d \log d - d \mathcal{V}(g)$
故 $\mathcal{V}(g) = \log d - N_{GCFT}(g)/d$。                □

注：卷一正文将 $N_{GCFT}(g) = \sum_i \lambda_i \log \lambda_i$
标注为"von Neumann 熵负值"是不精确的——严格地说，标准 von
Neumann 熵是 $-\sum_i p_i \log p_i = \mathcal{V}(g)$，而
$N_{GCFT}(g) = d \log d - d \mathcal{V}(g)$ 是与 von Neumann 熵
线性相关的"谱曲率项"（各向同性时为 0，各向异性时为正）。本卷
将价值 $\mathcal{V}$ 直接定义为标准 von Neumann 熵，避免符号
混乱。

物理含义：
    - $\mathcal{V}$ 度量认知系统的信息连通性
    - $\mathcal{V}$ 大 → 各维度信息流通畅（高连通）
    - $\mathcal{V}$ 小 → 各维度信息阻塞（低连通，高各向异性）
    - $\mathcal{V}$ 不显含 $\hbar$，是纯几何信息论量

定理 14.1（最大价值原理）
在共形约束 $\sum_i \lambda_i = d$ 下，$\mathcal{V}$ 的最大值在
所有 $\lambda_i$ 相等时取得：

    $\max \mathcal{V} = \log d$，当 $g = cI$（$c = 1$）

证明：由 Schur 凹性，对称凹函数在等分量时取得极值。$\mathcal{V}$
是 Shannon 熵形式，严格凹，故等分量 $\lambda_i = 1$（即 $g = I$）
时取得最大值 $\log d$。                                          □

推论 14.1（价值与空性同义反复）
"最大连通性"等价于"度规各向同性化"（卷一空性定义）：

    $\max \mathcal{V}$ ⟺ $g \to cI$ ⟺ 空性

价值不是空性的对立面，而是空性的同一面相的另一名称。这把
13.0 的"价值公理"还原为卷一已有的变分结构。

17.3  升维方案：Lawvere-Scott 自指函子塔极限

升维策略：将"层0元物理空间"重新表述为 Lawvere-Scott 自指函子塔

【概念澄清】
本卷构造的 LSSFTT 是 Lawvere (1970) 自指范畴理论与 Scott (1972)
反身域理论的范畴论构造，**不是**同伦类型论中的 ∞-groupoid
（Kan 复形）。两者在"无限自指"的哲学意义上相通，但数学构造
不同。本卷不再使用"∞-群胚"术语，避免概念混淆。

定义 17.1（Lawvere-Scott 自指函子塔）
设 $\mathcal{C}_0$ 为基础范畴（卷四的 0-截断），递归定义：

    $\mathcal{C}_{n+1} = \mathcal{C}_n^{\mathcal{C}_n}$

即 $\mathcal{C}_{n+1}$ 是 $\mathcal{C}_n$ 到自身的函子范畴
（幂范畴）。这是 1-阶自指的 n 次迭代。

【数值实现说明：幂范畴的笛卡尔积近似】
严格幂范畴 $\mathcal{C}_n^{\mathcal{C}_n}$ 的对象数为
$|\text{Obj}(\mathcal{C}_n)|^{|\text{Obj}(\mathcal{C}_n)|}$，
随 n 增长远快于指数。本卷的数值实现采用笛卡尔积近似
$\mathcal{C}_{n+1} \approx \mathcal{C}_n \times \mathcal{C}_n +
\mathcal{C}_n$（维度满足 $a_{n+1} = a_n^2 + a_n$），这是为了
数值可计算性的简化。在此近似下，$\rho_n = 1 - 2/\dim(\mathcal{C}_n)$
的收敛性是数值观察，严格的幂范畴收敛性需未来工作。这不影响
LSSFTT 的概念正确性——Scott 不动点定理在 DCPO 范畴中保证
连续自函子的最小不动点存在，只是本卷的数值实现是简化版本。

LSSFTT 极限：

    $\mathcal{O} = \lim_{n \to \infty} \mathcal{C}_n$

【边界声明：极限的范畴论含义】
此处 $\lim$ 的严格含义需要在 DCPO（定向完备偏序集）范畴中
形式化：连续自函子 $F: \text{DCPO} \to \text{DCPO}$ 总有最小
不动点（Scott 不动点定理）。本卷的数值观察（ρ_n → 1）与 Scott
不动点定理的预测一致，但**未形式化 DCPO 结构**——这是未来
严格化工作的方向。

性质：
    - 目标同构 $\mathcal{O} \cong \mathcal{O}^\mathcal{O}$
