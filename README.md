一、理论概述
------------------------------------------------------------------------

广义认知场论（GCFT）是一个以认知、意识、痛苦、修行、命运、
解脱等人类存在根本问题为研究对象的物理-数学理论框架。其核心
论断为：

    物理学之所以必须是「量子」的，是因为「非对易代数」是任何
    自指系统维持逻辑一致性的唯一稳定结构。

理论以两个基本自由度为第一性原理：

    κ（痛苦深度，duḥkha）——度量认知系统偏离基态的能量偏差；
    α（定力刚度，samādhi）——度量认知系统抵抗度规弯曲的弹性
                                 恢复能力。

一切其他物理量（度规张量 g、势能面 V、般若参数 δ、出离心 ρ、
意识整合度 Φ）均从 κ 和 α 推导，无任何额外硬编码参数。这是
理论可证伪性的根基。

理论由五卷本构成，遵循严格的截断塔（truncation tower）关系：

    卷一（经典动力学层）：单体认知流形上的变分原理
          核心成果：791:1 势垒比定理、三大不可逆奇点
    卷二（统计力学层）：多体网络与关系本体论
          核心成果：冻结业力定理、一阶觉照相变
    卷三（量子场论层）：度规量子化与本体论跃迁
          核心成果：瞬子顿悟、量子纠缠业力、涅槃不死定理
    卷四（元物理学层）：范畴论与数学起源
          核心成果：Lawvere 不动点定理应用、截断塔同伦收敛、
                    C*-代数唯一完备性、Topos 内蕴非对易性
    卷五（整体认知本体论层）：价值、基质、叙事与观测者悖论的极限解
          核心成果：价值势能场（von Neumann 熵）、全息界面方程
                    （扩展 Lindblad）、叙事流形（Berry 联络 Wilson
                    环路）、元物理极限（LSSFTT 自指函子塔极限）

理论累计完成 155/155 项验证全部通过（卷一至卷四 135 项 + 卷五
20 项），并提出 26 条可证伪红线（F1-F26），其中 9 条为数学证明
的必然性，1 条为数值验证，16 条为待检验的经验预测。

------------------------------------------------------------------------

       「为什么现有数学必须如此」。

       非从 κ、α 推导——这是诚实的边界声明。
    7. 卷五叙事流形 U_N 是 U(1) 简化实现，非阿贝尔 Berry 联络
